# Woolies-merchandise
# 🛒 Woolies Merchandise Sales Dashboard (Excel Project)

This project is a complete Excel-based Sales Dashboard created using the Woolies merchandise dataset.  
It includes data cleaning, transformation, pivot tables, and an interactive dashboard to analyze sales performance.

---

## 📁 Project Files
- **Woolies merchandise - Data.xlsx**  
  Contains the raw dataset used for cleaning and analysis.

- **Woolies merchandise - Solution.xlsx**  
  Contains the cleaned dataset, pivot tables, and the final interactive dashboard.

---

## 🎯 Project Objectives
- Clean and prepare the merchandise sales dataset  
- Build pivot tables for sales insights  
- Create an interactive dashboard for business decision-making  
- Analyze sales performance across:
  - Product categories  
  - Regions  
  - Time periods  
  - Customer segments
